# Sicentre

## L'agence
Sicentre est une agence digitale 360° avec IA, basée en Amérique Latine, qui transforme la présence en ligne des entreprises grâce à trois piliers : sites web sur mesure, agents vocaux IA, et production visuelle premium. Elle s'adresse aussi bien aux PMEs ambitieuses qu'aux grandes entreprises qui veulent franchir un cap.

## Services et tarifs

Sites web : plan Vitrina à $500 (paiement unique) et plan E-commerce à $1.000 (paiement unique).

Agente de Voz IA : plan Starter à $500 par mois, plan Business à $1.500 par mois, plan Enterprise sur devis.

Studio Visual : plan Starter à $500 par mois, plan Business à $1.500 par mois, plan Enterprise sur devis.

Plan ULTRA 360 : à partir de $3.000 par mois, sur devis, combinant l'ensemble des services.

## Identité visuelle

La direction artistique repose sur deux couleurs : un violet profond autour de #7C3AED et un orange vibrant autour de #EA580C. Tu peux ajuster les nuances pour servir le design.

Le fond global est sombre, jamais blanc pur ni noir pur. L'esthétique recherchée est celle du liquid glass : profondeur, transparence sophistiquée, lumière diffuse, glassmorphism poussé. Le site doit respirer le premium, le futurisme assumé, l'épure et la sobriété maîtrisée.

Pour la typographie, AUTOMATA-DISPLAY est disponible dans /public/fonts/ pour les titres principaux. Tu as toute liberté pour proposer une font body moderne, premium et cohérente avec l'ensemble. Si tu trouves une font display encore plus impactante que AUTOMATA, propose-la.

Les illustrations et visuels doivent être créés sur mesure pour chaque section. Privilégie les compositions SVG inline, les scènes graphiques originales, les visualisations abstraites cohérentes avec l'esthétique liquid glass. Évite les pictogrammes lucide-react posés tels quels comme illustration principale d'une card. Les icônes lucide ne sont autorisées que pour les éléments d'interface secondaires (flèches, boutons, listes).

## Langue

Tout le contenu est en espagnol rioplatense paraguayen. Vouvoiement systématique avec "usted". Les mots anglais universellement compris dans le tech sont acceptés (AI, web, studio, etc.).

## Ton éditorial

Le ton est posé, confiant, premium, jamais corporate ni générique. Les phrases sont courtes, percutantes, concrètes. On parle business, transformation, résultats, mais sans jargon marketing creux. Pas de superlatifs faciles (révolutionnaire, incroyable, le meilleur). Le client doit sentir qu'il a affaire à des pros qui savent ce qu'ils font, pas à une agence qui essaie de vendre. Chaque texte doit pouvoir être prononcé à voix haute sans gêne.

## Stack technique

Next.js 14 avec App Router, TypeScript, Tailwind CSS, Framer Motion.

## Outils à utiliser librement

Tu disposes du MCP 21st.dev Magic. Pioche dedans systématiquement quand un composant peut être enrichi par une référence existante de qualité.

Tu disposes du skill UI UX Pro Max. Applique-le à chaque décision visuelle : typographie, espacements, hiérarchie, palette, mise en page.

Tu disposes de Framer Motion. Anime intelligemment, pas pour le show. Les animations doivent être fluides, subtiles, premium.

## Approche de construction

Tu construis le site page par page dans cet ordre : landing, voz-ia, web, motion, precios, nosotros, contacto. Tu peux ajouter une page supplémentaire si elle apporte une vraie valeur.

Pour chaque page, tu utilises ton meilleur jugement UX/UI. Le contenu doit être concret, cohérent, jamais générique. Les exemples fictifs du portfolio doivent être nouveaux, impactants, crédibles pour le marché latino.

## Règles fermes

Pas de cards génériques avec pictogrammes plats. Chaque illustration doit être pensée, créée, intégrée au design global.

Pas de petites étiquettes ou labels au dessus des titres de section. Les titres se suffisent à eux mêmes.

Pas de mention de l'origine géographique de l'agence dans le hero ou les sections principales. Cette information vit uniquement dans le footer si nécessaire.

Pas de promesse de délai chiffré.

Pas de fond blanc pur.

Le responsive est mobile first. Breakpoints : 375px, 768px, 1280px.

npm run build doit passer proprement avant qu'une page soit considérée comme terminée.

## Esprit du projet

Liquid glass, animations soignées, premium, efficacité, sobriété, épure, design naturel et jamais générique, cohérence visuelle d'une page à l'autre, contenu très concret. Le site doit donner envie de cliquer, de lire, de contacter.
